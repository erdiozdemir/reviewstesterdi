# ReviewsInBio

ReviewsInBio is a lightweight desktop application for browsing the most recent
scientific articles from PubMed in major biomedical and bioinformatics fields.

The application is designed for students and researchers who want to quickly
explore up-to-date literature through a simple graphical interface.

---

## 🧪 Supported research fields

- 💻🧬 Bioinformatics
- 🦠 Microbiology
- 🧬 Genetics
- 🛡️ Immunology
- 🎗️ Cancer research

---

## ✨ Features

- Simple login screen
- Field-based article browsing
- Real-time PubMed queries (NCBI E-utilities)
- Displays the 10 most recent articles
- Clickable links to PubMed
- Clean and minimal desktop UI (Tkinter)
